公司汇报日报周报月报后台管理系统WorkReport

# 概述

项目是基于Net6.0+SQL Server进行开发的框架。
整合应用技术包括

# 后端

Asp.NetCore MVC、EF、Autofac、Log4Net。

# 前端

Layui为主。

# 核心功能

实现了基础**登陆鉴权**、**异常日志**中间件与增删改demo、**读写分离**模块、**Service自动注入**、公共返回类库等。
